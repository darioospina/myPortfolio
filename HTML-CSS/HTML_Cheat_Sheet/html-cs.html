HTML Basics
Structure

The basic HTML structure contains the following elements:

  • <HTML> ⇒ This is the container of all the HTML elements.
  • <HEAD> ⇒ Contains the metadata (data about data). All the information written here will not be displayed.
  • <TITLE> ⇒ Title to be displayed in the browser's title bar.
  • <BODY> ⇒ Contains all the elements that will be displayed in the website.

Images

The tag <IMG> is used to add images to the website

Within the tag, there are some properties that must be included for the image to be displayed:

  • Property src: This stands for source and asks for the url or location of the image. If the image is not stored in your PC, you must include within quotation marks the url of the image. For example, the picture of garfield, displayed to the right is external, therefore, the tag must call the image from the url that stores it. <IMG src="https://pixy.org/1246721/">. On the other hand, the picture below garfield was stored in my PC, therefore the tag I used to display it was <IMG src="cat1.jpg">
  • Property alt: This stands for alternative and means a text to be displayed if the image can't load or if it no longer exists.
  • Property width: This property defines the width of the picture. CSS can be used to add style to the image or define any property you want to change, however if you want a shortcut just for changing the size, you can use width.

Combining all the properties, the final <IMG> tag should be something like this <IMG src="cat1.jpg" alt="garfield" width="300px">

Styling

For adding style to the HTML elements there is CSS (standing for Cascading Style Sheets). This is a styling language that can provide styling and design to any HTML element.

There are two ways for linking CSS with HTML:

  • The first one is by adding the <style> tag.
  • The second one is by linking the HTML file with a CSS file. For doing so, the following tag can be used <link href="CSS_File_Name.css" rel="stylesheet">

In both cases, within each HTML tag you are adding style, you must include either a class or an ID. These are references to the HTML tag you want to call from the CSS. For instance, I'm going to add some style to the following <div> tag.

<div class="addingStyle"></div>
